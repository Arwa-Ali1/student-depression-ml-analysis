# 🧠 Student Depression Prediction (ML Project)

This project applies machine learning techniques to predict student depression based on lifestyle, academic, and psychological factors.

## 📌 Dataset
- Student Depression Dataset (Kaggle)
- ~27,900 records
- 18 features (academic, lifestyle, mental health)

## 🤖 Algorithms Used
- Logistic Regression
- Neural Network (MLP)
- Naïve Bayes
- Support Vector Machine (SVM)

## ⚙️ Tools
- Weka
- RapidMiner

## 🧪 Data Preprocessing
- Removing irrelevant attributes (ID)
- Handling missing values
- Normalization
- Encoding categorical data

## 📊 Results
- Best Accuracy: **85.33% (Logistic Regression - RapidMiner)**
- Neural Network: ~83.5%
- Naïve Bayes: ~85%
- SVM improved after fine-tuning

## 📈 Key Insights
- RapidMiner performed better after tuning
- Naïve Bayes not sensitive to tuning
- Neural Networks benefit from normalization & tuning
- Logistic Regression was the most stable model

- Data preprocessing importance
- Hyperparameter tuning
- Working with real datasets
